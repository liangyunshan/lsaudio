lsaudio
=======

lsaudio

Like unix "ls" command, but also display additional information for audio files, like duration, bitrate, size.
Output is easily parseable. o

Fields are like so:
size/minutes/seconds/bitrate/filename 

Example:
```
$ lsaudio
```
